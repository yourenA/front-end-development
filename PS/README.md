# photoshop 相关

## PSD自动标注工具 PxCook
http://www.fancynode.com.cn/pxcook
* 通过拖拽点选，标注自动生成
* 点击即生成前端代码


## ps 辅助线插件 guideguide
https://guideguide.me/

将解压后的压缩包放在C:\Program Files\Adobe\Adobe Photoshop CC 2018(安装目录)\Plug-ins文件内


## 快捷键
* 同时按下ctrl+delete可以用背景色填充
* 同时按下alt+delete可以用前景色填充
* alt+; 显示与隐藏快捷键

## 其他操作
* 效果->叠加 可以将图片/文字不透明的地方改变颜色
* 吸管工具里面又标尺，可以测量距离
